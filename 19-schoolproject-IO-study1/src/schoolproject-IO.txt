**School Project IO version**

기존 Map 과 Exception 을 적용했던 School Project에 IO (입출력) 를 적용해 프로젝트의 기능을 업데이트한다 

- 요구사항 1 
  사용자가 콘솔 상에서 데이터를 입력하면 그 데이터를 입력받아
  학교 구성원 정보를 관리할 수 있도록 기능을 추가한다 
  주요 기능은 콘솔 상에서 학교 구성원 정보를 유형별(학생,교사,직원)로 
  등록받을 수 있고 전화번호로 구성원 정보를 검색 및 삭제할 수 있다 
  또한 전체 구성원 리스트를 출력할 수 있다 
  대안 흐름에 관련해 
  검색 및 삭제시 입력한 전화번호가 존재하지 않는 번호이면 
  콘솔 상에 사용자에게 메세지를 전달한다 ( 실행화면.txt 참고 )
  
  -----------------------------------------------------------------
  19일 여기부터 시작 ConsoleInstUI6
  
  또한 등록시 전화번호가 중복되면 메세지를 전달해 다시 입력받도록 한다
  구성원 등록(addView())
  1. 입력 구성원 종류를 잘못 입력했을 때 다시 입력받게한다
  2.  등록시 전화번호 입력받으면 그 즉시 중복여부를 확인해 중복될 경우 다시 전화번호를 입력받게 한다 
  
 - 요구사항 2 
  사용자가 입력한 학교 구성원 정보를 지속적으로 유지한다 
  프로그램 종료시 객체 직렬화하여 파일에 저장하고 
  프로그램 시작시 객체 역직렬화하여 파일에서 다시 로드한다
